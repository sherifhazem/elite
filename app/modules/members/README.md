# Members Module

- **Routes:** Member-facing pages and APIs are defined in `app/modules/members/routes/` and `app/modules/members/auth/`.
- **Services:** Member utilities and role helpers live in `app/modules/members/services/`.
- **Forms:** Authentication and profile forms are under `app/modules/members/forms/`.
- **Templates:** Member portal templates reside in `app/modules/members/templates/`.
- **Static:** Member CSS and JS assets are under `app/modules/members/static/`.
